# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly:

1. **Do NOT** open a public issue
2. Email: knowledgeknight99@protonmail.com
3. Include a description and steps to reproduce
4. Allow 48 hours for initial response

## Scope

This is an OSINT intelligence toolkit. Security concerns include:
- API key exposure in configuration files
- Unsafe data handling or storage
- Injection vulnerabilities in search queries
- Docker container security issues
